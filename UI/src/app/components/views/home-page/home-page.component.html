Architecture Refactoring Helper

The Architecture Refactoring Helper is an application which aims to help find suitable migration approaches to microservices architecture.

Add a new refactoring approach!

Add a new refactoring approach that is not yet represented in the repository.

 Explore existing refactoring approaches!

Freely explore all refactoring approaches that exist in the repository. No recommendation for suitability for your application will be made. If that is something you are interested in check out the 'Find refactoring approaches for your application!' card.

 Find refactoring approaches for your application!

Find recommended refactoring approaches based on preferred inputs, outputs, processes and usability aspects of the approach.
